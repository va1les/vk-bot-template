# install
### 📨 **File** `example.env` rename to `.env`
```Shell
token=VkBotToken
uri=MongoUri
apiVersion=# https://dev.vk.com/reference/versions
```

### 📥 Install all modules — `npm install`

### ✅ Launch the Bot — `node .`
