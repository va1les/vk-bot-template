# Установика
### 📨 **Файл** `example.env` переменуйте в `.env`
```Shell
token=VkBotToken
uri=MongoUri
apiVersion=# https://dev.vk.com/reference/versions
```

### 📥 Установите все модули — `npm install`

### ✅ Запустите бота — `node .`
